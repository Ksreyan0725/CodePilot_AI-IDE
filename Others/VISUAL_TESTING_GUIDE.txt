================================================================================
COMMON ISSUES TO WATCH FOR
================================================================================

1. HORIZONTAL OVERFLOW:
   - Symptom: Page wider than screen, can scroll left/right
   - Check: Long email address, form padding, navbar elements
   - Fix applied: word-break: break-all, reduced padding

2. CONTENT CLIPPING:
   - Symptom: Text or elements cut off at edges
   - Check: Container padding, element margins
   - Fix applied: padding: 0 12px on container at 380px

3. OVERLAPPING ELEMENTS:
   - Symptom: Elements appear on top of each other
   - Check: Navbar items, form fields
   - Fix applied: Reduced gap between navbar items

4. UNREADABLE TEXT:
   - Symptom: Text too small or too light
   - Check: Font sizes, color contrast
   - Fix applied: Minimum 14px font size, proper contrast

5. UNUSABLE BUTTONS:
   - Symptom: Buttons too small to tap
   - Check: Touch target sizes
   - Fix applied: Minimum 44px or acceptable 28-34px for icons

6. BROKEN LAYOUT:
   - Symptom: Elements stack incorrectly
   - Check: Flexbox/grid behavior
   - Fix applied: Single column at 380px, proper flex properties

7. PERFORMANCE ISSUES:
   - Symptom: Laggy animations, slow scrolling
   - Check: Backdrop blur intensity, animation complexity
   - Fix applied: Reduced blur to 8px, simplified transitions
